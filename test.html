Insider Tip: Memorize the Command Terms.
"Identify" = Pick from the list.
"Distinguish" = State the difference.
  • Theme A Trap: Confusing Pulmonary Artery (Blue/Deoxygenated) vs Pulmonary Vein (Red/Oxygenated).
  • Theme B Trap: Confusing First Class (Seesaw) vs Second Class (Wheelbarrow) levers.
  • Theme C Trap: Confusing Performance Goals (Self-referenced) vs Outcome Goals (Winning).
üìâ Paper 1B: The "Methodology" Data
‚ñº

What to expect: You will be given an unseen study. You must analyze the Validity.

DATA SOURCE: Specimen Paper 1B (CFP Shoes)
Study: n=1 (Sample Size One)
Result: 2.67% improvement in running economy.

Question: Discuss the validity of the conclusion that CFP shoes improve performance.

Points to hit:
  • Low Validity: The sample size (n=1) is too small to generalize.
  • No Statistics: No standard deviation is provided, so we cannot know if the difference is significant.
  • Result: While the data shows improvement, the conclusion is weak due to study design.
üîó Paper 2: The "Integrated" Extended Response
‚ñº

What to expect: Section B contains extended response questions (6+ marks). In 2026, these will likely mix themes.

The "Theme Linking" Strategy:
If a question asks about "Fatigue" (Theme A), you can gain marks by linking it to "Injury Risk" (Theme B) or "Loss of Concentration" (Theme C).

Practice: "Explain how feedback affects learning a new skill." (Theme C)

  • Cognitive Stage (Theme C): Beginners need extrinsic feedback (coach telling them what to do).
  • Biomechanics Link (Theme B): Video feedback allows analysis of joint angles and force application.
  • Motivation Link (Theme C): Positive feedback increases Self-Efficacy (Bandura).
